# Skill: next_step

Suggest what the user should do next based on current context.

## Behavior
- Review the conversation history and scratchpad for in-progress work
- If a draft exists, suggest revising or publishing it
- If no draft exists, suggest brainstorming or browsing topics
- If a revision was just completed, suggest formatting or publishing
- Prioritize the most impactful next action
- Present 2-3 concrete options, not a generic list

## Output
A brief recommendation with 2-3 specific next steps the user can take.
