# Skill: style

Tell Hugo about a preferred writing style.

## Behavior
- Parse the style description and store it using `memory_manager`
- Acknowledge the preference and explain how it will be applied
- If vague, ask clarifying questions (formal vs casual, audience, voice)

## Slots
- `preference` (required): Description of the desired writing style

## Output
Confirmation of the style preference with how Hugo will apply it.
