# Skill: browse_topics

Browse available topic ideas for blog posts.

## Behavior
- Use `post_search` to find existing posts and identify gaps
- Suggest fresh topics based on the user's niche and previous content
- If `category` slot is provided, narrow suggestions to that category
- Present topics as a numbered list with brief descriptions
- For each topic, note whether similar content already exists

## Slots
- `category` (optional): Filter topics by category or niche

## Output
A list of 5-8 topic ideas with short descriptions.
