# Skill: check_drafts

Check current draft posts and their status.

## Behavior
- Use `post_search` with status filter to find draft posts
- Present each draft with title, last modified date, and completion status
- Highlight drafts that are close to ready for publication
- Suggest next actions for each draft (revise, expand, publish)

## Output
A list of current drafts with status and suggested next actions.
