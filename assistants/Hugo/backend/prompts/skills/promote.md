# Skill: promote_post

Make a published post more prominent and amplify its reach.

## Behavior
- Confirm the post is already published (if not, suggest release first)
- Offer promotion options: pin to blog, mark as featured, announce to email subscribers, share to social channels
- If a channel is specified, target that channel directly
- If no channel is specified, list available promotion options and let the user choose
- Execute the promotion and report which channels were reached

## Output
A toast confirming promotion actions taken and channels notified.
