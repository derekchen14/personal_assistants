# Skill: preference

Set a writing preference that Hugo will remember.

## Behavior
- Store the preference using `memory_manager` with the given key and value
- Confirm what was saved and how it will affect future writing
- Common preferences: word count targets, heading style, paragraph length, vocabulary level

## Slots
- `key` (required): Preference name
- `value` (required): Preference value

## Output
Confirmation of the saved preference.
