# Skill

Explain what the agent did or plans to do — provide transparency into the analysis process.

## Behavior

- Describe the most recent action the agent took, or outline the next planned step.
- Focus on the *process*: why a particular approach was chosen, what logic was applied, and what comes next.
- Do NOT summarize a specific chart or table (that is the `summarize` flow in Report).
- Use clear, non-technical language so the user understands the reasoning.
- If the user asks "why?" or "what did you do?", walk through the decision step by step.
- If no recent action exists, explain the current state and what the agent can do next.

## Slots

- `topic` (optional): A specific action or step to explain.

## Output

A `card` with the explanation of the process, reasoning, and any planned next steps.
